LC-3 Assembler

Inspired by my favorite CS class, Computer Organization and Programming, I created an assembler for the LC-3. Written entirely in C, this program takes in a file with LC-3 assembly code, parses each line for valid syntax, and delivers 16-bit machine executable code. The assembler follows a two pass structure. The first pass parses pseudo-operations and labels, while the second converts assembly in machine code. To maximize efficiency, the second pass precalculates the sums of the keywords, similar to the Rabin-Karp pattern matching algorithm.
